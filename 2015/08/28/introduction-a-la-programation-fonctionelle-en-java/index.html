Introduction à la programmation fonctionelle en java

28 August 2015 on java, google guava, java 8, stream API, programmation fonctionnelle

Dans cette article, une petite introduction à la programmation fonctionelle en Java.
Parce qu'un exemple est plus parlant qu'un long texte attaquons de suite :

Cas d'étude :
Dans cette exemple très simpliste, on souhaite faire une méthode qui à partir d'une liste de personne, nous renvoie la liste des prénoms des personnes ayant plus de 18 ans.
On présentera un exemple de code permettant de faire cela :

  • Avec une boucle for et un if imbriqué (méthode "normale")
  • Avec google guava (pour Java < 8)
  • Avec l'API de Stream (Java JDK 8)

Modèle de donnée : 

public class Person {

    private String name;

    private String surname;

    private int age;

}

A partir de cette classe on peut voir qu'une personne à donc un nom (name), un prénom (surname) et un age (age).
Par soucis de lisibilité les getters et setters seront omis.

Méthode "normale" : 

public class IterativePersonService {

    public List<String> getPersonNameWithAgeSup18(List<Person> persons) {

        List<String> result = new ArrayList<String>();

        for(Person person : persons) {
            if(person.getAge() > 18) {
                result.add(person.getName());
            }
        }

        return result;
    }
}

Un cas assez simple, on itère sur la liste de personnes et pour chaque personne, si son age est supérieur à 18, on ajoute son nom à une liste de nom que l'on retourne en résultat de notre méthode.

Avec google guava (pour Java < 8) : 

public class GuavaPersonService {

    // Static Predicate and Function can be externalized
    public static final Predicate<Person> AGE_SUP_18 = new Predicate<Person>() {
        public boolean apply(Person person) {
            return person.getAge() > 18;
        }
    };
    public static final Function<Person, String> GET_NAME = new Function<Person, String>() {
        public String apply(Person person) {
            return person.getName();
        }
    };

    // Method for introduction to functional programming begin here
    public List<String> getPersonNameWithAgeSup18(List<Person> persons) {

        public List<String> getPersonNameWithAgeSup18(List<Person> persons) {

        List<String> result = FluentIterable.from(persons) // Get FluentIterable from list of persons
                .filter(AGE_SUP_18) // after this step only person with age > 18 remains
                .transform(GET_NAME) // tranform list of person into list of person.name
                .toList(); // return list

        return result;
    }
}



Avec l API Stream (pour Java 8 +) : 

public class StreamPersonService {

    public List<String> getPersonNameWithAgeSup18(List<Person> persons) {

        List<String> result = persons.stream() // Use streamAPI
                .filter(p -> p.getAge() > 18) // for each person p, keep if p.age > 18
                .map(p -> p.getName()) // for each person p, return p.name
                .collect(Collectors.toList()); // return list of p.name

        return result;
    }
}
